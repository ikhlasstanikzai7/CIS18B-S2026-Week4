# Mini-Assignment 5: Debugging an Exception

## Objective
Use the debugger to identify and fix a bug that causes an exception.

## Problem Statement
A bug in an alert-processing loop causes the application to crash. Instead of guessing, use the debugger in VS Code or GitHub Codespaces to inspect variables and determine what went wrong.

## Requirements
- Run the program in debug mode.
- Set a breakpoint in the loop.
- Identify the exact line that causes the exception.
- Fix the bug.
- Write a short text file named `debugging-notes.txt` explaining:
  - what the bug was
  - how you found it
  - what change fixed it
